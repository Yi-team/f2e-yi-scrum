# 作品說明
本網頁的設計宗旨是讓 Scrum 新手可以透過像遊戲一樣的方式來理解整個 Scrum 的運作原理，把從新手村到老手村的過程，使用 Scrum 的方式來規劃並到達目的地，每個角色各司其職，用幾個 Scrum 的週期去完成目標，跟著我們一起邊跑邊學習 Scrum 吧！
